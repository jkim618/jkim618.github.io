Sequeer

A B2C mobile app that serves as a comprehensive global search engine for the LGBTQ community

Overview

How do we design a search engine that enables users to locate LGBTQ-friendly establishments?

Problem

The company's new partner and creative director has identified that the current app's UX and UI are suboptimal. The poor design and functionality have resulted in a decline in user engagement. Specifically, users have found the app and website to be overly complicated and cluttered, making it difficult to navigate and locate desired features.

Solution

A thorough evaluation was conducted utilizing Jakob Nielson's 10 usability heuristics, and the app was redesigned to remedy the identified issues.

User Flows

To help guide the team in iterating through different features, I have created 3 key user flows for the following user stories:

  1. As a user, I want to be able to award badges to businesses.
  2. As a user, I want to be able to search businesses and look at their reviews and details.
  3. As a user, I want to be able to filter my search for businesses.
